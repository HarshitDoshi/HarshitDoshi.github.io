Mac and Breeze

How to cook up a beautiful desktop setup inspired by Apple's Mac OS and using KDE's wonderful Plasma desktop environment.

by Harshit Doshi

Posted on January 5, 2019 at 7:00 PM

I have to confess something: I am a big fan of Apple's design. I love their software design and their hardware design too. I find it harmonious, cozy, inviting, and well built.

I know that it is not perfect, that it has it's own flaws too, that there are a lot of people who find it repulsive, who don't find it intuitive, or who despise and hate it. But, this is about me. For me, it balances productivity and beauty. I find myself having a pleasant workflow when I am using a design like Apple's.

But, here's a shocker. I have never used any Apple products ever. I have never used Mac OS (except when I tried building a hackintosh, back in the Mac OS X Mountain Lion era).This may seem as fanboy-ism, or a wannabe, but I assure you that it isn't. Also, if you don't like Mac OS or Apple's design, you can very well stop reading here, because the next part of this blog post is all about how to replicate Mac OS's design and workflow on KDE Neon (based on Ubuntu) operating system which comes with KDE's Plasma desktop environment.

So, let's dive in!

I will assume that you have a computer running a fresh installation of KDE Neon. Our first step then, would be to update the whole system to the latest version. To do so, one needs to open a terminal window. You can find the terminal under the System category in the applications menu named as Konsole. Open it and type in the following commands:

sudo apt-get update && sudo apt-get upgrade

After that command finishes running and you are taken back to the prompt on the terminal, you may exit the terminal. You then need to open Discover, the graphical software manager made by KDE. Open Discover and go to the updates section. You may find updates which are not covered by the terminal commands here. Update them all and close Discover.

Our next step then, would be to install a dock. The people at KDE have made a wonderful dock called Latte. Install Latte dock by opening the terminal as described in the previous step and type in the following:

sudo apt-get install latte-dock
